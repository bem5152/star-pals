# Star Pals — Rocket Adventure

A toddler-friendly, Galaga-style auto-shooter built for Makenna (age 3). A googly-eyed rocket
auto-fires upward; the only input is **dragging left/right** to steer. Endless, celebratory,
and hard to lose — clouds → butterflies → star-creatures, with a friendly boss every 3rd level.

**Play it:** https://bem5152.github.io/star-pals/

Single self-contained file — `index.html` (Canvas 2D + Web Audio API, zero dependencies,
no asset files). Built for iPad Safari: open the link, then **Share → Add to Home Screen**
for a fullscreen app icon. The first tap unlocks sound (an iOS requirement).

## Deploy
Static site on GitHub Pages — push to `main` and Pages auto-redeploys. No build step.
Vercel works too (point it at this repo; framework preset "Other").
